# ArcSDE_ManageTools
ArcSDE数据库优化管理工具
## 使用说明

1. 工具通过*.sde数据库连接文件连接数据库，使用前确认./bin 目录下存在正确的数据库连接文件；
2. 直接双击 **1.数据库优化工具箱.bat** 文件运行工具；

## 主要功能
                                   
           A. 重建数据库索引;                B. 重建统计信息;               
           C. 压缩版本库;                    D. 断开全部SDE连接;
